ME-471 - Advanced Computer Aided Engineering

The class involved working in a group to create an optimized model of a remote control car. We created a skeleton model and used top-down modeling in order to create a fully parameterized model in NX. The work included measuring the physical model of the car, creating the necessary parameters, modeling the solid parts and shell surface using the parameters,, performing topology optimization, finite element analysis, and computational fluid dynamics analysis.
